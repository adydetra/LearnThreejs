## Get Started

Clone repository

```bash
git clone https://github.com/adydetra/LearnThreejs.git
cd LearnThreejs
```

Install package manager

```bash
bun install
```

Run the app in `development` mode.

```bash
bun dev
```

---

> [!NOTE]
> if not using `bun` just delete `bun.lockb`, and switch to another package manager like:

```bash
# npm install
# yarn install
# pnpm install
```
